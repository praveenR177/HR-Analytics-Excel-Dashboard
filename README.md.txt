# HR Analytics Dashboard – Excel

## Project Overview
This project analyzes employee attrition using Excel. 
The dashboard helps HR teams identify attrition trends across departments, job roles, age groups, gender, marital status, and education fields.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- KPI Metrics

## Key Insights
- Sales and R&D departments show higher attrition
- Employees aged 26–35 have the highest attrition
- Certain job roles contribute more to attrition
- Certain Education Fields are increased in attrition 

## Conclusion
This Excel dashboard provides interactive and data-driven insights to support HR decision-making and employee retention strategies.
