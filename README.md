About This Project:

I analyzed an HR employee dataset to understand employee attrition patterns and workforce trends.
Using Microsoft Excel, I performed exploratory data analysis (EDA) with pivot tables, charts, and slicers to identify key factors influencing attrition such as department, job role, age group, gender, marital status, and education field.
The goal of this project was to help HR teams make data-driven decisions to improve employee retention.
